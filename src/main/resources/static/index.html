Welcome to Bookstore

This is the home page of your Spring Boot application. The API is secured with JWT; use the endpoints under /api for programmatic access.

Refresh Login (API) 
Available endpoints
POST /api/auth/register
POST /api/auth/login

GET  /api/v1/users/me   (role: USER)
GET  /api/v1/users      (role: ADMIN)
POST /api/v1/users      (role: ADMIN)
Tech stack

Spring Boot 3, Spring Security (JWT), JPA, PostgreSQL.

How to use

Register and login via API to receive a JWT, then call protected routes with the token.
